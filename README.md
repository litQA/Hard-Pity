# 大保底

**大保底 \(Hard Pity\)** 是一款**生存类 Meme 模组**，旨在优化《我的世界》中稀有物品的获取体验，为**凋灵骷髅头颅、三叉戟、龙蛋、下界合金碎片**添加了保底掉落机制，解决玩家刷取稀有物品时的 “脸黑” 痛点。

---

## 卡池抽奖系统

### 触发方式

每当玩家击败以下四种生物时，会在对应卡池进行**1 次基础抽奖**：

- 凋灵骷髅 → 凋灵骷髅卡池

- 溺尸 → 溺尸卡池

- 猪灵 / 猪灵蛮兵 → 猪灵卡池

- 末影龙 → 末影龙卡池

**额外抽奖规则**：

- 主手工具 / 武器有**抢夺附魔**时，根据等级额外进行**抽奖**

- 每个卡池的累计抽数**独立计算**

### 卡池奖励详情

|卡池|四星奖励|五星奖励 \(不歪 / 大保底\)|五星奖励 \(歪\)|
|---|---|---|---|
|凋灵骷髅|凋灵玫瑰|**凋灵骷髅头颅**|骷髅头颅|
|溺尸|铜锭|**三叉戟**|钻石剑|
|猪灵 / 猪灵蛮兵|金锭|**下界合金碎片**|棕色染料|
|末影龙|龙首|**龙蛋**|鸡蛋|

## 概率与保底规则

### 五星奖励 \(稀有物品\) 保底机制

模组支持自定义保底抽数设置，玩家可在配置文件中自由调整保底数值，模组默认保底抽数为40抽，当单卡池累计抽数达到设置的保底数时，必定触发五星稀有物品掉落，百分百出金。同时模组提供三种不同的抽卡概率模型，玩家可按需切换，默认采用mihoyo模型，各模型具体概率规则如下：

#### mihoyo模型（默认）

整体概率机制贴合mihoyo抽卡规则，保底前最后5抽概率大幅递增，出金概率分别为16%、33%、51%、80%、100%。保底倒数第5抽之前的所有抽数，单抽出金概率固定为 $0.21 \div (保底次数-5)$，概率节奏平稳，临近保底概率大幅飙升。

#### 线性模型

采用均匀递增的概率机制，每一抽的出金概率遵循固定公式：$(当前次数 + 1) \div 保底次数$。概率从0开始，随抽数增加匀速线性增长，直至保底次数时概率达到100%，整体概率增长规律直观、稳定。

#### 指数型模型

概率增长曲线更为平滑温和，单抽出金概率计算公式为：$0.3679^{(保底次数 - 当前次数)} - 1$。相较于另外两种模型，该模式下前期概率缓慢提升，全程无概率突变，抽卡出货节奏更加平缓。

### 小保底与歪率机制

- 默认小保底**不歪概率为50%**，该概率数值支持在配置文件中自定义修改，可自由调整出货正歪比例。

- 若本次出金为 “歪”，则**下一次出金必定为目标五星**\(大保底\)，随后重置计数

### 四星奖励保底机制

- **独立保底：默认20发**，该保底数值可在配置文件中自定义调整，四星保底计数与五星保底计数相互独立、互不影响。

- 同样受抢夺附魔影响，可获得额外抽奖机会。

## 捕获明光机制

该机制以 NBT 形式储存在玩家数据中，所有卡池共享一套计数器，计数器初始值为0，数值值域为\-1、0、1、2。

玩家触发小保底出货歪奖时，计数器数值\+1，触发小保底出货不歪时，计数器数值\-1，最低为\-1；当计数器数值为1时，玩家触发小保底将拥有6%的概率触发“捕获明光”效果，触发后直接判定为不歪且计数器重置为0，未触发则正常执行原有歪率判定；当计数器数值为2时，玩家的小保底将直接判定为不歪，同时计数器重置为0。在歪率为默认50%的前提下，长期游玩积累抽奖次数后，该机制可近似将玩家整体的小保底不歪概率提升至约54\.94%。
